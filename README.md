# 📊 HR Analytics Dashboard

## 🔹 Description
This project focuses on analyzing **employee attrition trends** and workforce data.  
SQL is used for data analysis and **Power BI** is used to build interactive dashboards for better decision-making.

---

## 🛠️ Tools Used
- **SQL**  
- **Microsoft Excel**  
- **Power BI**  

---

## 📈 Key Insights
  🔸 Higher attrition observed in certain departments  
  🔸 Salary impacts employee retention  
  🔸 Experience level affects employee turnover  

---

## 🚀 Features
  ✔️ KPI Tracking (**Attrition %, Total Employees**)  
  ✔️ Department-wise Attrition Analysis  
  ✔️ Interactive Power BI Dashboard  
  ✔️ Data Cleaning & Transformation  

---

## 📂 Project Files
- 📊 Dataset (CSV/Excel)  
- 📜 SQL Queries (.sql)  
- 📈 Power BI Dashboard (.pbix)  

---

## 🎯 Conclusion
This project helps in understanding key factors affecting **employee attrition** and supports **data-driven HR decisions**.
